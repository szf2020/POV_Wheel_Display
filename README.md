# POV_Wheel_Display
Persistence of Vision (POV) display based on the ESP32-S3. 
This project drives a 4-arm LED rotor (SK9822-A LEDs) and features a fully autonomous web interface for rendering static images and GIF animations, complete with advanced power management and wireless synchronization.
